## 实现注意事项

1. **时间排序逻辑**：需要将季度转换为数字（冬=1，春=2，夏=3，秋=4），然后按年份和季度组合排序
2. **折叠功能**：使用 useState 管理折叠状态，添加平滑的展开/收起动画
3. **时间轴设计**：使用绝对定位或 flex 布局实现时间轴，确保与卡片的正确对齐
4. **响应式设计**：在小屏幕上可能需要调整时间轴的显示方式

## 技术实现要点

- 使用 Tailwind CSS 实现时间轴和动画效果
- 保持代码结构清晰，遵循现有代码风格
- 确保功能的可维护性和扩展性
